内容
	一、Support功能范围
		1）请求第三方Client的组装；
		2）交易数据的交易及统一加加签名处理；
		3）交易请求封装；
		4）结果数据封装，将第三方交易数据转为对象，对外提供；
		5）异常处理，如支付宝支付交易异常，将发起一次交易查询；
	二、CxxxTxxServiceImpl
		1）负责将业务数据（Settlement）组装成请求数据，提供基础配置数据；
		2）只管调用专项交易接口及跟进结果返回修改业务数据（Settlement）
